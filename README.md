# Инструкция по установке
**Backend**
1. Перейдите в папку backend
  cd backend
2. Установите зависимости
  pip install -r requirements.txt
3. Получите API Key: Зарегистрируйтесь на openweathermap.org и получите ключ. Вставьте его в backend/main.py вместо "key".
4. Запустите сервер
   python main.py
