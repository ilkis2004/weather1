# Инструкция по установке
## **Backend**
1. Перейдите в папку backend: cd backend
3. Установите зависимости: pip install -r requirements.txt  
4. Получите API Key: Зарегистрируйтесь на openweathermap.org и получите ключ. Вставьте его в backend/main.py вместо "key".
5. Запустите: python main.py

## **Frontend**
1. Перейдите в папку frontend: cd frontend
2. Установите зависимости: npm install
3. Запустите: npm run dev
